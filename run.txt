üìù SETUP INSTRUCTIONS

Backend:
1. cd backend
2. npm install
3. Create .env file with:
   - MONGO_URI=your_mongodb_connection_string
   - JWT_SECRET=your_secret_key
   - PORT=3000
   - FRONTEND_URL=http://localhost:5173
4. npm start

Frontend:
1. cd frontend
2. npm install
3. Create .env file with:
   - VITE_API_URL=http://localhost:3000
4. npm run dev
